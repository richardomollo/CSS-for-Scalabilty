Open source project structuring Sass for scalability by using and extending ITCSS. Making CSS scalable and maintainable is a major concern for every front-end developer and large IT companies. I am currently combining SASS and extending ITCSS to create a scalable and maintainable SASS CSS starter template that will allow for easy management of CSS at scale.
